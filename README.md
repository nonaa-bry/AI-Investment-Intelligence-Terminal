# AI-Investment-Intelligence-Terminal

A multi-agent AI-powered investment research platform that provides company analysis, market intelligence, financial insights, and automated reports.

## Live Demo
https://ai-investment-intelligence-terminal-dvm7qzzvmkza5tfqnqww69.streamlit.app/

## Features
- Multi-Agent AI Architecture
- Fundamental Analysis
- Market Research
- Investment Insights
- PDF Report Generation
- Interactive Dashboard

## Tech Stack
- Python
- Streamlit
- Gemini/OpenAI
- Financial APIs
- Pandas
- Plotly
